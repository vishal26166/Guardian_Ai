 GuardianAI

**AI-powered proactive safety system for children and women**

 Overview

GuardianAI is an intelligent safety solution designed to enhance the protection of children and women through real-time location tracking, AI-based risk detection, and automated emergency alerts. The system proactively identifies potential threats and responds instantly without requiring manual intervention.

 Features

* 📍 Real-time location tracking
* 🧠 AI-based risk detection (Safe / Medium / High)
* 🚨 Automatic alert generation on high risk
* 📩 Instant notification system (simulated)
* 🔐 Secure backend with user authentication
* 🌐 Simple frontend dashboard for live monitoring
